## Overview
This project uses the **BRISC2025 Brain Tumor MRI Dataset**, which contains
T1-weighted brain MRI images for multi-class tumor classification.

Due to dataset size and licensing restrictions, the actual MRI images are
**not included** in this repository.

---

## Dataset Statistics
- **Total samples:** 6,000 images
- **Training set:** 5,000 images
- **Test set:** 1,000 images
- **Classes (4):**
  - Glioma
  - Meningioma
  - Pituitary Tumor
  - No Tumor
- **Imaging modality:** T1-weighted MRI
- **Planes:** Axial, Coronal, Sagittal (balanced representation)
- **Annotation quality:** Reviewed and corrected by medical experts
